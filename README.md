# 🚀 Project Overview

This project demonstrates how raw transactional data can be transformed into **actionable business insights using data visualization**.

The dashboard analyzes:

* Revenue performance
* Expense distribution
* Profit trends
* Regional performance
* Customer segment contribution
* Product line sales
* Payment method usage

The goal is to provide **decision-makers with quick insights** into business operations.

# 📑 Dataset Description

The dataset contains **transaction-level business data**.

| Column           | Description                    |
| ---------------- | ------------------------------ |
| Transaction_ID   | Unique ID for each transaction |
| Transaction_Date | Date of the transaction        |
| Revenue          | Total revenue generated        |
| Expenses         | Cost incurred                  |
| Profit           | Revenue - Expenses             |
| Category         | Business category              |
| Region           | Geographic region              |
| Department       | Business department            |
| Product_Line     | Product type                   |
| Customer_Segment | Customer classification        |
| Payment_Method   | Payment type                   |
| Discount         | Discount applied               |

# 🔄 Data Processing Steps

### 1️⃣ Data Cleaning

* Removed missing values
* Verified transaction IDs
* Standardized categorical values
* Validated profit calculations

```
Profit = Revenue - Expenses
```

---

### 2️⃣ Data Preparation

* Structured dataset for analysis
* Ensured numeric consistency
* Converted date formats

---

### 3️⃣ Data Aggregation

Using **Pivot Tables** to analyze:

* Revenue by Region
* Profit by Category
* Sales by Product Line
* Revenue by Customer Segment
* Payment method distribution

Insights:
